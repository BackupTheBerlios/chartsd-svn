chartsd - Chart Daemon

What is it?

chartsd is a daemon, written by Alexander 'dma147' Mieland in the C programming language, which generates flow-charts diagrams from any log-file which has a specific format. It can be used to generate diagrams of the statistical usage of the temperatures of the cpu, mainboard, harddiscs or whatever. It also can be run more than once to generate more than one diagram of more than one source log-file.
View the [screenshots] page for information on how it can look like.
